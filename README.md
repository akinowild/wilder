# wilder
自已常用的一些函数方法的集合
《除夜》
> 扫除茅舍涤尘嚣，一炷清香拜九霄。
> 万物迎春送残腊，一年结局在今宵。
> 
> 生盆火烈轰鸣竹，守岁筵开听颂椒。
> 野客预知农事好，三冬瑞雪未全消。

野客：村野之人，多借指隐逸者。
所以偏了个词 wild+er=wilder

# 安装
```
pnpm add wilder
```
如果报TS的错误，则在src下面新建一个index.d.ts文件，在里面输入
```
declare module 'wilder';
```
# 引用文件

```
import wilder from 'wilder';
```
---------------------------------
下面的每一个方法的使用都是wilder.function名称
